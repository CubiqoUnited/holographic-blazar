# Holographic Blazar

This is a new project initialized via Antigravity.
It is connected to GitHub and deployed on Vercel.

## Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/CubiqoUnited/holographic-blazar.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run development server:**
   ```bash
   npm run dev
   ```
